== Description ==

[PREMIUM VERSION LINK](http://plugins.infosofttech.com/products/woocommerce-inventory-management/)

WooInventory a WooCommerce Stock Manager help you manage your stock related functionalities easily.

You can set Manage stock, Stock status, Backorders, Stock, Price, Sale price, Weight, Edit SKU, goto Product Edit page by clicking 'Edit' link for each product and variation.

**Dashboard**:
Dashboard shows various summaries - Low Stock Product, Out Of Stock Product, Most Stock Product, Zero Stock Product, Total Products, Total Simple Products, Total Variable Products, Total Variation Products.

**Features in  Pro Version**:
Paid version is True Inventory Manager, Features like Backdated stock position, Stock ledger, Stock Valuation, Stock Planner, Stock Alerts and much more 
[Click here](http://plugins.infosofttech.com/products/woocommerce-inventory-management/) to know more about Pro Version

Plugin is compatible with WooCommerce 3.4+ and is tested on 3.4.1 version.

== Installation ==

* Install the plugin from the 'Plugins' section in your dashboard (Go to `Plugins > Add New > Search` and search for 'WooCommerce Tax Report').
* Alternatively, you can [download](http://downloads.wordpress.org/plugin/wooinventory-lite.zip "Download WooCommerce Colored Order Notes") the plugin from the repository. Unzip it and upload it to the plugins folder of your WordPress installation (`wp-content/plugins/` directory of your WordPress installation).
* Activate it through the 'Plugins' section.
* Access the settings page from WordPress backend under WooInventory.
